GMathLibrary
======================

A mathematical overhaul for the garrysmod source implementation of vectors and rotations.

This will include the following features:
- Quaternions.
- Angles.
- Transforms.
- Vector2,3,4's.
- Standard Format.
- Documentation and Examples.
